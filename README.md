# Banking-Analysis
-end-to-end data analysis project using only excel.

📌 Project Overview:
The Banking Customer & Operations Analytics Dashboard is a data-driven analytics project designed to monitor and analyze key banking operations. It provides actionable insights into customer behavior, account performance, transactions, loans, card usage, and customer support efficiency.

🎯 Objectives:
The system is built to:
1-Track customer growth and churn risk
2-Analyze account balances and activity
3-Monitor transaction trends and detect anomalies
4-Evaluate loan portfolio performance
5-Measure card issuance and lifecycle trends
6-Assess customer support effectiveness
